# LEAGUE RADAR

# Project description
## Sports website where users can browse teams of their favourite sports and view games currently happening. 
## Users can also view the stats of teams and their schedule, they can also click a checkbox on the team schedule to get notified when the game starts.
## Users can also generate a random matchup from the available teams

# Instructions to run the website locally
## To start run the index.html,
## From Here Create account in top right, use unique name and email to create account
## After account creation go to sign in, and enter details, then return to index page and refresh to display user details
## Use drop down menu to view teams and games
## From games click button to generate random matchup
## From the teams pages use checkbox to get notified when a game will start, 
## (NOTE: THIS WORKS WITH REAL TIME, BUT TO TEST CHANGE VARIABLES MANUALLY IN scripts.js, under teams section)

# Known limitations
## The main limiaions of this website came from time constraints and include the following
## Not all teams and sports are available
## Matches are not updated live 
## Users can sign in but not do anything
